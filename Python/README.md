# XMLParser

#### Request json body structures
```javascript
{
    "profile_name":"your config file name",
    "params":[
        {"name":"tag  name",
         "value":"tag value"
        },

         {"name":"tag name",
         "value":"tag value"
        }
    ]
}
```

#### for example
```javascript
{
    "profile_name":"raouf",
    "params":[
        {"name":"debug",
         "value":"dada"
        },

         {"name":"key3",
         "value":"rr"
        }
    ]
}
```
